# House Price Prediction

## Description
This project is part of my Task 01 during my Machine Learning Internship at Prodigy InfoTech. The aim of this project is to build a simple model that can predict house prices based on input features.

## Dataset
train.csv

The dataset contains housing-related information used to train the model.

## Tools Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Scikit-learn  

## Method
A basic machine learning approach is used to estimate house prices from the given inputs.

## Steps Followed
- Loaded the dataset  
- Selected relevant features  
- Processed the data  
- Applied a machine learning model  
- Generated predictions  
- Visualized the results  

## Result
The model predicts house prices based on input values and demonstrates a basic implementation of regression using machine learning.
